Enhancing seismic site response analyses: tuning soil properties via genetic algorithms and Bayesian model updating from downhole array data

Shima Sadeghzadeh, Sberna A.P., Fabio Di Trapani, Cristoforo Demartino
Soil Dynamics and Earthquake Engineering — November 2024
PDF DOI Source

Abstract

This study employs deterministic genetic and probabilistic Bayesian algorithms to enhance seismic site response simulations through refined characterization of soil properties using downhole array data.

Seismic site responseDownhole array dataModel updatingGenetic algorithmBayesian model updatingUncertainties quantification

Highlights

  • Developed a 3D FE model for seismic site response; improved soil accuracy.
  • Used genetic algorithms to tune shear-wave speeds to match seismic records.
  • Used Bayesian updating to quantify soil-property uncertainty; more reliable.
  • Validated on 2011 Kütahya and 2012 Ege Denizli quakes; robust, applicable.
